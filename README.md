# clap-info-rs

> A simple command line CLAP information tool in Rust

⚠️ Disclaimer: This is a Rust port of [clap-info](https://github.com/free-audio/clap-info/) made for the solely purpose of learning [CLAP](https://github.com/free-audio/clap), [clack](https://github.com/prokopyl/clack) and Rust together.
